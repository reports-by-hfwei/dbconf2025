		if it can be \red{$\wrdep$-extended} to a \purple{\it full history} which satisfies $\mathcal{M}$.
	\end{definition}

		with $\ge \rc$ and a fixed number of $\ge 3$ sessions

		even \blue{irrespectively} of the complexity of these predicates.
